心晴2020

发表于 2020-08-29 更新于 2020-10-17 分类于 note

2020年

8月29日

  彩虹已逝，佳人未归

9月2日

  为了避免引起怀疑
  我们有时会有一些争执
  但是那些是出于爱，而不是出于不满
  它们不代表愤怒，而是代表了
  一种最甜蜜的温柔

  生日快乐，我对自己说，生日要快乐~

9月3日

  期待着你在生日尾巴的亲口祝福，可惜没有。

  你的抱歉，不能让我快乐。

9月6日

  硬硬的烤牛肉🍖，能喝的麻辣烫🍅，好好吃的绵绵冰🍨

  一般般的烤腿🍗，不如自己煎的牛排🥩，美味的水果茶🍹

  just so so 的瓦罐汤🍲，很下饭的青椒炒肉🌶️，好难喝的蜜雪冰城⛄

  美丽的公园🏞️，悠闲的开车车🏎️，刺激的坐车车😨,精彩的马戏🐴

  短暂而又愉快的周末，还想和你吃好多好吃的，玩很多刺激的，不对，刺激的就少玩点吧😅

  匆匆的别离，因为害怕，眼泪它不争气🥺

  感谢再一次的相遇，让我们好好珍惜❤

9月12日

  喜得佳人归，胜似迎新年

  虽然还要出差- -

9月14日

  终于有了早点下班的动力了🤗

10月13日

  泡温泉♨️，穿和服👘，吃美食😋，放烟花🎆，游泳🏊‍♀️，一封不知能否寄出的信💌

  与你在一起的时间，总是过得很快很快，今天你又要出差了，出差顺利~

  我会想你，很想很想你。这次可能是我不适应了吧😂

  注意休息，多喝水，多喝水，多喝水 : ) 重要的事情说三遍！
